# ¿Qué es Termux?

Termux es un emulador de terminal para Android y un entorno Linux que funciona sin necesidad de root. Trae consigo un sistema de paquetes básico que permite acceder a herramientas de desarrollo, editores de texto, compiladores y miles de aplicaciones listas para usar.

⚡

Sin Root

Funciona sin necesidad de rootear tu dispositivo, mantendo la seguridad de Android.

📦

Gestor de Paquetes

APT y pkg para instalar aplicaciones Linux de forma sencilla y actualizada.

🔧

Entorno Completo

Acceso a GCC, Python, Ruby, Node.js, Git, vim, emacs y mucho más.

🔐

Seguridad

Entorno aislado que no requiere permisos especiales ni root.

☁️

SSH/SCP

Conectate a servidores remotos directamente desde tu Android.

🐍

Python Included

Python preinstalado con pip listo para usar desde el primer momento.

# Personalización

Archivo .bashrc

Personaliza tu terminal con aliases y configuración.

# ~/.bashrc
alias ll='ls -la'
alias la='ls -a'
alias ..='cd ..'
alias update='apt update && apt upgrade'

PS1='\[\033[01;32m\]\u@\h\[\033[00m\]:\[\033[01;34m\]\w\[\033[00m\]\$ '
export EDITOR=nano

Prompt con Colores

Un prompt más visual y útil.

PS1='\[\033[01;32m\]\w\[\033[00m\]\[\033[01;36m\]\$\[\033[00m\] '
# Verde para directorio
# Cyan para $

Oh My Zsh

Instala zsh con plugins y temas.

apt install zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

Powerlevel10k

Tema moderno para zsh.

git clone --depth=1 https://github.com/romkatv/powerlevel10k.git $ZSH_CUSTOM/themes/powerlevel10k
# Edita ~/.zshrc: ZSH_THEME="powerlevel10k/powerlevel10k"

Neofetch

Muestra info del sistema con estilo.

apt install neofetch
neofetch

Plugins Útiles

Mejora tu experiencia con plugins.

# En ~/.zshrc
plugins=(git docker python node npm)
source $ZSH/oh-my-zsh.sh

# Distros Linux en Termux

Instala distribuciones Linux completas dentro de Termux usando proot-distro.

¿Por qué instalar una distro?

  • ✅ Acceso a más paquetes y herramientas
  • ✅ Entorno más similar a un PC Linux
  • ✅ Distintas versiones de distribuciones
  • ✅ Aislamiento del sistema base
Distro Tamaño Mejor Para Complejidad
Ubuntu ~100MB Desarrollo general, Docker Baja
Debian ~90MB Estabilidad, servidores Baja
Arch Linux ~80MB Rolling release, personalizado Alta
Fedora ~110MB Innovación, containers Media
Kali Linux ~120MB Seguridad, pentesting Media
Alpine ~50MB Ligero, containers Media

Instalación de Ubuntu (ejemplo)

1

Instalar proot-distro

apt install proot-distro
2

Listar distros disponibles

proot-distro list
3

Instalar Ubuntu

proot-distro install ubuntu
4

Iniciar sesión en Ubuntu

proot-distro login ubuntu
5

Actualizar e instalar herramientas

apt update && apt upgrade
apt install git curl wget vim python3 build-essential

Instalación de Kali Linux

# Instalación de Kali Linux
proot-distro install kali

# Iniciar Kali Linux
proot-distro login kali

# Dentro de Kali - actualizar
apt update && apt full-upgrade

# Instalar herramientas Kali
apt install kali-linux-default

Instalación de Arch Linux

# Instalación de Arch Linux
proot-distro install arch

# Iniciar Arch Linux
proot-distro login arch

# Dentro de Arch - actualizar
pacman -Syu

# Instalar base
pacman -S base-devel git curl wget

# Entorno Gráfico (GUI)

Añade un escritorio gráfico a tu distro Linux en Termux mediante VNC.

🎯 Requisitos Mínimos

  • Dispositivo con Android 7.0+
  • 2GB de RAM disponible
  • 4GB de almacenamiento libre
  • Una distro Linux instalada

📱 Apps Necesarias

  • Termux (ya instalado)
  • Termux:Widget (opcional)
  • VNC Viewer (RealVNC o similar)
  • Una distro con proot-distro
🖥️

Xfce

Ligero y rápido. Ideal para dispositivos con recursos limitados.

RAM: 512MB Espacio: 1GB
🎨

LXQt

Moderno y ligero. Alternativa a Xfce con look más actual.

RAM: 512MB Espacio: 1GB
🔵

MATE

Clásico y estable. Fork de GNOME 2.

RAM: 1GB Espacio: 2GB
⚡

KDE Plasma

Completo y moderno. Necesita más recursos.

RAM: 2GB+ Espacio: 3GB

Guía de Instalación: Xfce + VNC

1

Instalar servidor VNC en la distro

# Entra a tu distro
proot-distro login ubuntu

# Instala XFCE y VNC
apt update && apt install xfce4 xfce4-goodies tightvncserver
2

Configurar VNC

# Configurar VNC server
vncserver

# Contraseña: ingresa una contraseña (6-8 caracteres)
# Verás preguntas, responde 'n' a las opciones de viewonly
3

Configurar inicio de sesión

# Crear archivo de configuración
mkdir -p ~/.vnc

# Editar configuración
vim ~/.vnc/xstartup
4

Contenido de xstartup

#!/bin/bash
xrdb $HOME/.Xresources
startxfce4 &
5

Hacer ejecutable y reiniciar VNC

chmod +x ~/.vnc/xstartup
vncserver -kill :1
vncserver :1 -geometry 1280x720 -depth 24
6

Instalar termux-x11 (opcional, para mejor experiencia)

# En Termux (host), NO en la distro
apt install termux-x11

# En la distro, instalar cliente X11
apt install xorg-xauth
7

Conectar desde VNC Viewer

# Configuración en VNC Viewer:
# Address: localhost:5901
# Nombre: Ubuntu XFCE

# O si es externo (misma red):
# Address: 192.168.x.x:5901

Método Alternativo: termux-x11 (Sin VNC)

# 1. Instalar en Termux host
apt install termux-x11

# 2. En la distro, ejecuta:
export DISPLAY=:0
termux-x11 :0 &

# 3. Inicia las apps que quieras
xfce4-session &

# Workstation Móvil Linux

Los tres entornos de trabajo que puedes tener en tu Android.

Nivel 1

Termux Puro

📟

Entorno base de Termux sin distros adicionales.

Características

  • ✅ Acceso directo a sistema Android
  • ✅ Paquetes optimizados para Termux
  • ✅ Ligero y rápido
  • ✅ Funcionalidad completa

Casos de Uso

  • Scripts de automatización
  • Desarrollo Python/Node básico
  • SSH a servidores
  • Aprendizaje de comandos Linux
Nivel 2

Distro CLI

🖥️

Distribución Linux completa en modo terminal.

Características

  • ✅ Sistema Linux tradicional
  • ✅ Acceso a paquetes disto-específicos
  • ✅ Aislamiento del sistema host
  • ✅ Múltiples distros disponibles

Casos de Uso

  • Servidores locales
  • Compilación cruzada
  • Contenedores Docker
  • Kali Linux tools
Nivel 3

Entorno Gráfico

🖼️

Escritorio Linux completo con interfaz gráfica.

Características

  • ✅ Escritorio tradicional
  • ✅ Aplicaciones GUI
  • ✅ Navegador web completo
  • ✅ Ofimática básica

Casos de Uso

  • Trabajo de oficina móvil
  • Desarrollo IDE completo
  • Edición de imágenes
  • Acceso remoto a PC

🎯 Recomendación Final

Para una workstation móvil efectiva, te recomendamos:

  1. Inicia con Termux puro - Aprende los comandos básicos y familiarízate con el entorno.
  2. Instala una distro CLI - Ubuntu o Debian para tener acceso completo a herramientas.
  3. Añade GUI solo si es necesario - Xfce para equipos con recursos limitados, o KDE para equipos potentes.

# Datos Curiosos

01

Sin Play Store

Termux nunca ha estado en Google Play Store debido a políticas de la tienda. Se distribuye exclusivamente en F-Droid y GitHub.

02

Basado en Ubuntu

Los paquetes de Termux están construidos sobre la base de Ubuntu 22.04 (Jammy Jellyfish), lo que garantiza compatibilidad.

03

No requiere Root

A diferencia de otras soluciones, Termux funciona completamente sin permisos de root, usando namespaces de Linux.

04

10,000+ Paquetes

El repositorio de Termux ofrece más de 10,000 paquetes precompilados listos para instalar.

05

Comunidad Activa

La comunidad de Termux es muy activa en GitHub y Telegram, con más de 50,000 estrellas en el repositorio.

06

Multi-Terminal

Puedes ejecutar múltiples sesiones de terminal simultáneas, ideal para administración de sistemas.

07

SSH Inverso

Puedes configurar SSH inverso para acceder a tu Android desde cualquier lugar del mundo.

08

API Android

Termux tiene una API que permite acceder a sensores, cámara, SMS y otras funciones de Android desde la terminal.

# Preguntas Frecuentes

No, Termux funciona completamente sin permisos de root. Utiliza Linux namespaces para crear un entorno aislado dentro de Android. Esto mantiene la seguridad del dispositivo intacta.

Sí, Termux puede usarse para hacking ético y pruebas de seguridad. Instala Kali Linux con proot-distro para acceder a herramientas como Nmap, Metasploit, etc. Siempre usa estas herramientas de manera legal y con autorización.

Sí, puedes ejecutar servidores Nginx, Apache, o incluso servicios como MySQL, PostgreSQL, Redis, etc. Es perfecto para desarrollar y probar proyectos web desde tu móvil.

Ejecuta termux-setup-storage para dar permisos. Esto creará una carpeta ~/storage con enlaces a Downloads, DCIM, etc.

En Termux puro no es posible instalar Docker completo debido a limitaciones del kernel. Sin embargo, en distros como Ubuntu o Debian puedes usar Docker. Alternativamente, usa containers con proot o namespaces.

pkg es un wrapper de Termux que simplifica la instalación. Internamente usa apt, pero pkg maneja automáticamente la configuración de repositorios.

No, Termux es exclusivo de Android. Para iOS existen alternativas como iSH (basado en Alpine Linux) disponible en la App Store.

Usa termux-backup para crear un backup y termux-restore para restaurarlo. El backup incluye todos los paquetes instalados y configuraciones.
